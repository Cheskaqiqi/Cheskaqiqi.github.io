A Red black tree is a self-balancing binary search tree, a data structure used in computer science.

It was invented in 1972 by Rudolf Bayer and called symmetric binary B-trees. Later, in 1978, it was modified by Leo. guibas and Robert sedgewick into the current Red-Black Tree R-B Tree, the full name is red-black tree, also known as “red-black tree”, which is a special binary search tree. #definition:

A red-black tree is a balanced binary search tree in which the order traversal is monotonically undiminished
Nodes are red or black.
The root node is black.
Each leaf node (also known as an external node, which is intended to turn a red-black tree into a true binary tree, i.e. a NULL node, an empty node) is black.
The two children of each red node are black. (In other words, there can be no two consecutive red nodes on all paths from each leaf to the root.)
All paths from the root node to each leaf contain the same number of black nodes (this value is called the black height).
